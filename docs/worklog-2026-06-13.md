## 做了什麼

### 1. MLB 日報搬上 GitHub Actions
- 為 poly-smartmoney 建私人 repo `vinnymacc/poly-smb`、接 git
- `.github/workflows/mlb-report.yml`：cron 排程，每天 **3 次**（台灣 00:10 / 08:30 / 17:00）
- TG token / chat id 放 GitHub Secrets（`.env` 不進 repo）
- 跑在 GitHub 伺服器 → **家裡電腦關機也會發**
- 刪掉本機重複的兩個日報排程（避免重複發），本機只留 15 分鐘線

### 2. 聰明錢名單
- 新增 4 個地址（老虎/野狼/犀牛/海豚），全 MLB tag
- watchlist 共 23 個地址，無重複
- 維護 `config/animals.md` 動物名稱池（120+，已用的標記）

### 3. 15 分鐘線排版
- 新增 `describeBet()`：把盤口翻成看得懂的描述
  - 勝負盤 → 隊名
  - 大小分 → `大小分 9.5 Under`（不再是光一個 Under）
  - 讓分 → `讓分 主-1.5 隊名`
- `Bet` 加 `slug` 欄位以支援上述描述

### 4. 修 bug
- **賽事日期判定錯誤**：原本用「台灣日期減 1」推算美國賽事日期，算出沒有比賽的日期 → 誤判「無賽事」不發。改成抓 **slug 日期 ≥ 今天的最早那批**，三個時段都鎖定同一批賽事，不受排程延遲 / 跨午夜影響。
- **大回應解析 bug**（稍早）：`http.ts` 的 chunked 解碼改用 Buffer，修好大筆 JSON（如賽程 130KB）解析失敗。
